A central dashboard for Store One modules including Woo Search, Smart Cart, Frequently Bought Together, and more WooCommerce enhancements.

== Description ==

Store One is an extendable WooCommerce enhancement toolkit that brings all important features under a single unified dashboard.

Instead of installing multiple heavy plugins, Store One offers modular components that you can enable/disable as needed.

Current modules include:
- **Smart Woo Search** – Live Ajax search with beautiful layouts
- **Smart Cart** – Enhanced cart drawer and cart experience
- **Frequently Bought Together** – Show product bundles and combos
- **Pre-Orders Module** – Allow pre-ordering products with custom rules
- More modules coming soon…

Each module is optimized for performance, easy to configure, and fully integrated inside the Store One admin UI.

### Key Features:
- Unified dashboard to manage all modules
- Toggle modules on/off
- Lightweight & modular architecture
- Clean React-based admin interface
- Compatible with all WooCommerce themes
- Developer-friendly & extendable

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/store-one` **or** install it directly from the Plugins screen.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Go to **Store One** in your WordPress admin sidebar.
4. Enable the modules you want and configure their settings.

== Frequently Asked Questions ==

= Is Store One compatible with any theme? =
Yes. It works with all WooCommerce-compatible themes, including Astra, Storefront, GeneratePress, Block-based themes, and more.

= Do all modules load if I activate the plugin? =
No. Store One loads **only** the modules you enable. This keeps things fast and lightweight.

= Does this plugin slow down WooCommerce? =
No. The plugin is modular and optimized to load only required scripts.
